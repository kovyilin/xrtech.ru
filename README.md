# XRTECH.RU Landing Page

Современный одностраничный макет сайта, посвящённого тематике **XR (Extended Reality)** — виртуальной (VR), дополненной (AR) и смешанной (MR) реальности.

Проект выполнен в минималистичном технологическом стиле с использованием чистых HTML, CSS и JavaScript без сторонних библиотек.

## Особенности

- Адаптивный дизайн
- Cyber-tech оформление
- Canvas-анимация на главном экране
- Плавные анимации появления элементов
- Стеклянные (glassmorphism) карточки
- Поддержка мобильных устройств
- Без зависимостей и фреймворков

## Используемые технологии

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API

## Назначение

Макет может использоваться как основа для:

- информационного портала о XR-технологиях;
- корпоративного сайта компании, работающей с VR/AR/MR;
- презентационного сайта проекта в сфере Spatial Computing;
- образовательного ресурса об иммерсивных технологиях.

## Статус проекта

Данный репозиторий содержит демонстрационный шаблон одностраничного сайта, который может служить отправной точкой для дальнейшего развития полноценного веб-проекта.

## Лицензия

Проект распространяется в ознакомительных целях. Перед использованием сторонних изображений, логотипов или материалов рекомендуется убедиться в соблюдении условий их лицензирования.
